# 🦈 SharkSkills

A curated vault of **1,350+ AI agent skills** for use with [Google Antigravity](https://deepmind.google/technologies/antigravity/) and compatible agentic coding assistants.

## What is this?

SharkSkills is a massive, production-grade collection of specialized skills that supercharge your AI coding assistant. Each skill is a structured `SKILL.md` file (sometimes with supporting guides, scripts, and examples) that teaches the agent expert-level knowledge in a specific domain.

## How to Use

### With Antigravity

1. Clone this repo into your Antigravity skills directory:
   ```bash
   git clone https://github.com/joaovitorbelortee-cpu/sharkskills.git
   ```

2. Point your Antigravity skill orchestrator to the `skills/` folder.

3. Use the **Antigravity Skill Orchestrator** to autonomously select and apply the best skill for any task.
